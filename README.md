# Telegram Sales Bot (Arabic)

برنامج تيليجرام لتسجيل المبيعات وإدارة المنتجات باستخدام SQLite.

## المتطلبات
- Python 3.8+
- الحزم:
  - python-telegram-bot==13.15

تثبيت الاعتمادات:
```bash
pip install -r requirements.txt
```

## الإعداد
- أنشئ متغير بيئة للتوكن:
```bash
export TELEGRAM_BOT_TOKEN="YOUR_BOT_TOKEN"
```

- شغّل البوت:
```bash
python3 bot.py
```

سيتم إنشاء قاعدة البيانات `sales.db` تلقائيًا في نفس المجلد.

## أوامر البوت
- للأعضاء:
  - `/start`: رسالة ترحيبية.

- للإدمن فقط (حدّث قائمة `ADMINS` في `bot.py`):
  - `/help`: عرض جميع الأوامر.
  - `/add_product <الاسم> <السعر> <المخزون>`: إضافة منتج.
  - `/list_products`: عرض المنتجات.
  - `/set_price <id> <السعر>`: تعديل سعر منتج.
  - `/set_stock <id> <الكمية>`: تعديل مخزون منتج.
  - `/sale <id> <الكمية>`: تسجيل بيع لعنصر واحد.
  - `/sale <id1>:<q1>,<id2>:<q2>`: بيع متعدد العناصر.
  - `/today_sales`: تقرير مبيعات اليوم.
  - `/report <YYYY-MM-DD> <YYYY-MM-DD>`: تقرير لفترة زمنية.

## ملاحظات
- قاعدة البيانات SQLite مضمنة ولا تتطلب إعدادًا إضافيًا.
- تأكد من وضع التوكن كمتغير بيئة وعدم تخزينه داخل الكود.
